# pydxf2dwg 1.0

pydxf2dwg is a small program in Python 3.11 which convert dxf to dwg.



## Installation

   - You have to install pyautocad

```
pip install pyautocad

```

## Requirements

  Only for Windows

  Autocad must be installed

## Usage

  - Start Autocad.

  - Otherwise Autocad will start.

  - Start pydxf2dwg

  - Choose .dxf files

  - Wait a minute
